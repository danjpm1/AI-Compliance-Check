Build an EU AI Act Compliance Checker web app for my portfolio targeting AI engineering jobs in Germany.

================================================================================
PROJECT OVERVIEW
================================================================================

A tool where companies input information about their AI system and receive:
1. Risk classification based on EU AI Act categories (Unacceptable, High-Risk, Limited, Minimal)
2. Legal obligations specific to their category
3. Compliance checklist with EU AI Act article references
4. Downloadable PDF report

This is a portfolio project that recruiters will test, so it MUST work flawlessly, look professional, and demonstrate real value.

================================================================================
TECH STACK
================================================================================

Frontend:
- React 18+
- Tailwind CSS
- React Router for navigation
- Axios for API calls

Backend:
- Python Flask
- Flask-CORS
- OpenAI API (GPT-4) for generating explanations
- ReportLab for PDF generation

Deployment: Replit (must work on Replit's infrastructure)
